template<class T, std::size_t N, class Container = ::estd::declare::deque<T, N>>
class estd::declare::queue< T, N, Container >

Declaring a statically-sized queue object of size N.

Template Parameters
TThe type of the elements in this queue.
NThe maximum number of elements in the queue.
ContainerThe type of the underlying implementation container.
