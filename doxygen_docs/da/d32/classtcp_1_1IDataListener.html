◆ dataReceived()

virtual void tcp::IDataListener::dataReceived ( uint16_t  length)
pure virtual

Callback that gets called when data has been received.

Parameters
lengthNumber of bytes available to read.
Note
It is not required to read the data immediately from within this callback, the socket will store the data and the next dataReceived callback will have a larger length parameter. However, keep in mind that TCP has a WINDOW_SIZE. If data is not read from the socket and length becomes WINDOW_SIZE, not further dataReceived callback will occur because the sender cannot send any more data as the window size is zero.

Implemented in tcp::TcpIperf2Server, tcp::LoopbackTestServer, and tcp::BandwidthTestSocket.
