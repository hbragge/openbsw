Detailed Description

template<class T, class Comparator = std::less<T>>
class estd::ordered_vector< T, Comparator >

Implementation of a fixed-size ordered vector. Elements are inserted via the insert method. The Comparator object defines the sort order.

Template Parameters
TThe type of elements stored in the ordered vector.
ComparatorThe sorting order.

Memory Usage

The memory usage in bytes for this ordered_vector will be:

32 bit OS: N * sizeof(T) + 20 64 bit OS: N * sizeof(T) + 40

Note
It is possible there will be extra bytes used for proper memory alignment depending on your compiler settings.

Usage example

void insertRandom(estd::ordered_vector<int>& v)
{
for(int i = 0; i < v.max_size(); ++i)
{
v.insert((rand() * 50);
}
for(auto iter = v.begin(); iter != v.end(); ++iter)
{
printf("%d\n", *iter);
}
}
