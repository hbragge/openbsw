Detailed Description

template<class T, std::size_t N>
class estd::declare::deque< T, N >

deque with fixed size.

Template Parameters
TType of values of this deque.
NMaximum number of values in this deque.
Note
This container will allocate N + 1 elements, because the N + 1st element will be used for the end-iterator.
