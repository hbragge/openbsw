Detailed Description

Abstract base class for all sockets.

Overview

The AbstractSocket is designed to be a socket abstraction to a TCP stack. The socket class can be used to connect to a remote system. It is also returned to an ISocketConnectionListener by AbstractServerSocket (using its ISocketProvider) once a connection from a remote system is accepted. An active socket is used to receive and transmit data from and to a remote system.

Receive

The application that wants to receive data from a socket has to register through the setDataListener method as an IDataListener. After that the application is notified through its dataReceived method.

Send

Sending data through a socket is usually asynchronous. Because of that it is possible to register an IDataSendNotificationListener who is notified when the data passed to the send method is written to the TCP-stack. This is not the time when the ACK package from the remote system, is received!
