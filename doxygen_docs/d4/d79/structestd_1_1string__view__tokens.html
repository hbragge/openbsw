Detailed Description

Iterator over tokens of a string_view given a set of delimiters. Example:

string_view view{"Hello World"};
auto tokens view.tokenize(" ");
for (auto token : tokens)
{
cout << token << endl;
}
// stdout:
// Hello
// World
