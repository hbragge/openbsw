Detailed Description

template<class T, std::size_t N, class Container = ::estd::declare::vector<T, N>>
class estd::declare::stack< T, N, Container >

Type to declare a statically-sized stack object of size N.

Template Parameters
TThe type of the elements in this stack.
NThe maximum number of elements in the stack.
ContainerThe type of the underlying implementation container.
